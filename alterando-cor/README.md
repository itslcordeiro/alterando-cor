# 🎨 Alterando Cor de Fundo

Projeto web interativo para alterar a cor de fundo da página de diferentes formas.

## 📋 Descrição

Uma página simples que demonstra manipulação do DOM com JavaScript, permitindo ao usuário trocar a cor de fundo através de botões predefinidos, seletor de cor nativo do navegador ou digitando o nome/código hexadecimal de uma cor.

## 🚀 Funcionalidades

- Botões de cores predefinidas (Vermelho, Verde, Azul, Amarelo)
- Gerador de cor aleatória com valores RGB
- Campo de texto para digitar o nome ou código HEX da cor
- Seletor de cor nativo do navegador (`<input type="color">`)

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura da página |
| CSS3 | Estilização dos botões e layout |
| JavaScript | Manipulação do DOM e lógica de cores |

## 📁 Estrutura do Projeto

```
alterando-cor/
├── index.html      # Estrutura da página com botões
├── script.js       # Funções de troca de cor
└── css/
    └── style.css   # Estilos da aplicação
```

## ▶️ Como Executar

1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no navegador

## 💡 Conceitos Praticados

- Manipulação de DOM com `document.body.style`
- Geração de valores aleatórios com `Math.random()`
- Eventos de clique e mudança (`onclick`, `onchange`)
